var, let y const

Diferencias entre var-let y const en javascript
-var: Se puede reasignar y redeclarar en cualquier parte del código.
-let: Se puede reasignar, pero NO redeclarar.
-const: No se puede readsignar, ni redeclarar.

Operadores de Javascript

Tipos:

  • Asignación: Declarar y asignar variables (=)
  • Matemáticos: Para realizar cálculos (+,-,*,/,%,**)
  • Comparación: Booleanos (==, ===, !=, !==, >, <, >=, <=)
    • *Diferencia entre un operador débil y uno estricto (== vs ===): El operador débil toma en cuenta sólo el valor y no la naturaleza del mismo, es decir, no toma en cuenta el tipo de variable. Ejemplo: 5 == "5" → True, pero 5 === "5" → False
  • Lógicos: Permiten combinar condiciones. (&&, ||, !)
